# EchoMap

## Project Overview
EchoMap is an open-source, research-grade acoustic 3D mapping platform. It reconstructs indoor environments using only sound, acting as an acoustic equivalent of LiDAR.

## Features
- Acoustic 3D mapping using standard microphones and speakers.
- Signal generation (chirps, MLS).
- Cross-correlation and time-of-flight estimation.
- 3D point cloud generation.
